## История о прогнозе погоды

### Старый прогноз погоды vs Новый прогноз

**Старый способ (Точечный прогноз):**
```
Синоптик говорит: "Завтра будет +22°C"
```

Это всё. Одно число. Но насколько это полезно?

**Новый способ (Вероятностный прогноз):**
```
Синоптик говорит:
"Завтра СКОРЕЕ ВСЕГО будет между +20°C и +24°C
Наиболее вероятно около +22°C
Есть 20% шанс дождя"
```

Какой прогноз полезнее? Второй! Потому что теперь ты знаешь:
- МОЖЕТ быть прохладно до +20°C (взять лёгкую куртку?)
- МОЖЕТ быть тепло до +24°C (шорты подойдут!)
- Есть шанс дождя (может, взять зонтик?)

## Пример из жизни: Киоск с лимонадом

### Бизнес Маши по продаже лимонада

Маша продаёт лимонад по выходным. Ей нужно решить, сколько лимонада приготовить.

**Используя точечный прогноз:**
```
Папа говорит: "Ты продашь 50 стаканов завтра"

Маша делает ровно 50 стаканов.

Что произошло:
- Сценарий 1: Было очень жарко! 80 человек хотели лимонад.
              Маша всё продала и упустила возможную прибыль.

- Сценарий 2: Пошёл дождь! Пришло только 20 человек.
              Маша выбросила 30 стаканов лимонада.
```

**Используя вероятностный прогноз:**
```
Папа говорит: "Ты вероятно продашь от 30 до 70 стаканов.
               Скорее всего около 50 стаканов.
               Есть 30% шанс дождя (меньше продаж)"

Умное решение Маши:
- Сделать 40 стаканов для начала (безопаснее)
- Держать дополнительные ингредиенты на всякий случай
- Проверить погоду ещё раз утром
```

## Магия неопределённости

### Почему знать "Я не знаю" — это сила

Представь двух предсказателей:

```
Предсказатель А:
┌─────────────────────────────────┐
│ "Ты найдёшь ровно 1000 рублей  │
│  под кроватью завтра!"         │
└─────────────────────────────────┘
(Звучит уверенно, но скорее всего неправда)

Предсказатель Б:
┌─────────────────────────────────────────┐
│ "Я вижу деньги в твоём будущем...       │
│  Может быть от 200 до 2000 рублей,      │
│  скорее всего 500. Но есть и шанс       │
│  потерять 100 рублей, если не повезёт!" │
└─────────────────────────────────────────┘
(Честен насчёт неопределённости — гораздо полезнее!)
```

Второй предсказатель **вероятностный** — он говорит тебе:
- Что наиболее вероятно (найдёшь 500 рублей)
- Лучший случай (до 2000!)
- Худший случай (можешь потерять 100)
- Насколько он уверен (дал диапазон, а не точное число)

## Пример с трейдингом: Покупка биткоина

### Проблема одного числа

**Обычный прогноз:**
```
"Биткоин завтра будет стоить $50,000"

Трейдер думает: "Отлично! Куплю сейчас за $48,000 и заработаю $2,000!"

Но что если он упадёт до $45,000?
Трейдер потеряет $3,000!
```

**Вероятностный прогноз:**
```
"Биткоин завтра:
 - Наиболее вероятно: $50,000
 - Может упасть до: $45,000 (5% шанс)
 - Может вырасти до: $55,000 (5% шанс)"

Умный трейдер думает:
"Хм, я могу заработать $2,000-7,000...
 но могу и потерять $3,000.
 Инвестирую только сумму, которую готов потерять!"
```

## Картинка распределения вероятностей

### Как выглядит прогноз?

```
ПРОГНОЗ ОДНИМ ЧИСЛОМ:
                    │
                    │
                    ●  "$50,000"
                    │
                    │
    ────────────────┼────────────────
                   $50k

ВЕРОЯТНОСТНЫЙ ПРОГНОЗ:
                    │
                   ╱╲
                  ╱  ╲
                 ╱    ╲        <- Эта форма горы
                ╱  ●   ╲          показывает все
               ╱   │    ╲         возможности!
              ╱    │     ╲
    ─────────╱─────│──────╲─────────
           $45k  $50k    $55k

Высота показывает, насколько вероятна каждая цена:
- $50,000 наиболее вероятен (самая высокая точка)
- $45,000 или $55,000 возможны, но менее вероятны
```

## Квантили: Разрезаем гору на части

### Делим возможности на кусочки

Помнишь гору? Мы можем разрезать её на части:

```
           ╱╲
          ╱  ╲
         ╱    ╲
        ╱      ╲
       ╱        ╲
──────╱──────────╲────────
    │   │   │   │   │
   5%  25% 50% 75% 95%

5% линия:  "Только 5% шанс, что будет ниже этого"
25% линия: "25% шанс ниже, 75% шанс выше"
50% линия: "Пополам — МЕДИАНА"
95% линия: "Только 5% шанс, что будет выше этого"
```

Эти части называются **квантили**. Они помогают ответить на вопросы:
- "Что ХУДШЕГО может случиться?" (смотри на 5%)
- "Что ЛУЧШЕГО может случиться?" (смотри на 95%)
- "Что наиболее типично?" (смотри на 50%)

## Оценка: Был ли прогноз хорошим?

### Аналогия с мишенью для дартса

Представь, что бросаешь дротики в мишень:

```
Оценка точечного прогноза:
┌─────────────────────────────┐
│                             │
│         ○ ← Реальность      │
│                             │
│              ● ← Прогноз    │
│                             │
│    Оценка: Расстояние между │
│            ○ и ●            │
└─────────────────────────────┘

Оценка вероятностного прогноза:
┌─────────────────────────────┐
│                             │
│    ┌───────────────┐        │
│    │   ○ Реальность│        │
│    │   внутри      │        │
│    │   диапазона   │        │
│    │   прогноза!   │        │
│    └───────────────┘        │
│                             │
│    Оценка: Попала ли        │
│    реальность в диапазон?   │
│    Насколько узким был      │
│    диапазон?                │
└─────────────────────────────┘
```

Хороший вероятностный прогноз:
1. Реальный результат попадает в твой предсказанный диапазон
2. Твой диапазон максимально узкий (не жульничай, говоря "от $0 до $1 миллиона")

## Критерий Келли: Сколько ставить

### История про копилку

Представь, что у тебя 1000 рублей в копилке. Друг предлагает пари:

```
Пари:
- Подбросить монетку
- Если ОРЁЛ: Ты выигрываешь и получаешь УДВОЕННУЮ ставку
- Если РЕШКА: Ты теряешь свою ставку

Но подожди! Монетка друга ОСОБЕННАЯ:
- Она падает ОРЛОМ 60% времени!
- Она падает РЕШКОЙ только 40% времени!
```

**Вопрос:** Сколько из 1000 рублей ставить?

**Плохой ответ:** Поставить всё! (1000 рублей)
- Если проиграешь, у тебя 0 и ты больше не сможешь играть!

**Плохой ответ:** Ничего не ставить! (0 рублей)
- Ты упускаешь хорошую возможность!

**Умный ответ Келли:** Ставить 20% (200 рублей)
- Если выиграешь: 1000 + 200 = 1200 рублей
- Если проиграешь: 1000 - 200 = 800 рублей
- За много ставок это растит деньги быстрее всего!

```
Формула Келли (простая версия):
Ставка = (Шанс выигрыша × Выплата - Шанс проигрыша) ÷ Выплата
Ставка = (60% × 2 - 40%) ÷ 2 = 20%
```

## Калибровка: Честные прогнозы

### Аналогия с пожарной тревогой

Представь, что в школе есть пожарная тревога. Система должна быть **откалибрована**:

```
ХОРОШО ОТКАЛИБРОВАННАЯ ТРЕВОГА:
- Когда она говорит "90% шанс настоящего пожара"
  → Это действительно пожар в 90% случаев

ПЛОХО ОТКАЛИБРОВАННАЯ (Самоуверенная):
- Когда она говорит "90% шанс настоящего пожара"
  → Это действительно пожар только в 50% случаев
  → Ты перестаёшь доверять тревоге!

ПЛОХО ОТКАЛИБРОВАННАЯ (Неуверенная):
- Когда она говорит "10% шанс настоящего пожара"
  → Это действительно пожар в 60% случаев
  → Ты не готов, когда пожар случается!
```

Чтобы прогнозы были полезны, они должны быть **откалиброваны**:
- Когда говоришь "80% уверен" → Должен быть прав 80% времени
- Когда говоришь "20% уверен" → Должен быть прав 20% времени

## Принятие решений в неопределённости

### Дилемма магазина мороженого

Ты хочешь открыть киоск с мороженым, но не уверен, будет ли дождь:

```
БЕЗ вероятностного мышления:
"Погода говорит солнечно. Закажу 100 порций мороженого!"
→ Если пойдёт дождь, потеряешь всё.

С вероятностным мышлением:
"70% шанс солнца, 30% шанс дождя"

Рассчитай ожидаемый результат:
- Солнечно (70%): Продашь 100 порций × 100₽ прибыль = 10,000₽
- Дождь (30%): Продашь 20 порций × 100₽ прибыль = 2,000₽
- НО: Оставшееся мороженое стоит 50₽ за порцию

Ожидаемая прибыль = 0.7 × 10,000 + 0.3 × 2,000 - (расходы)

Умное решение: Заказать 60 порций
- Безопасное количество, которое приносит прибыль
  в ОБОИХ сценариях!
```

## Торговая стратегия

### Собираем всё вместе

```
Шаг 1: Получи вероятностный прогноз
┌─────────────────────────────────────────────┐
│ Прогноз на биткоин завтра:                  │
│ - 60% шанс, что вырастет                    │
│ - Ожидаемая доходность: +2%                 │
│ - Может упасть до -3% (худший случай)       │
│ - Может вырасти до +8% (лучший случай)      │
└─────────────────────────────────────────────┘

Шаг 2: Используй Келли для размера ставки
┌─────────────────────────────────────────────┐
│ Келли говорит: Ставь 15% капитала           │
│ Но мы осторожны: Ставим только 7.5% (половина)│
└─────────────────────────────────────────────┘

Шаг 3: Проверь, приемлем ли риск
┌─────────────────────────────────────────────┐
│ Если вложу 7.5% и случится худшее:          │

│ Это нормально — я могу позволить такую потерю│
└─────────────────────────────────────────────┘

Шаг 4: Совершаем сделку!
┌─────────────────────────────────────────────┐
│ КУПИТЬ биткоин на 7.5% капитала             │
│ Установить стоп-лосс на -3% (уровень худшего│
│   случая)                                    │
│ Надеемся на ожидаемый рост +2%!             │
└─────────────────────────────────────────────┘
```

## Почему это важно

### Сравним двух трейдеров

```
Трейдер А (Точечные прогнозы):
┌─────────────────────────────────────────────┐
│ "Модель говорит биткоин вырастет!"          │
│ → Вкладывает 50% сбережений                 │
│ → Биткоин падает на 5%                      │
│ → Теряет 2.5% всех сбережений               │
│ → Пугается, продаёт в убыток                │
│ → Модель оказалась права потом, но поздно!  │
└─────────────────────────────────────────────┘

Трейдер Б (Вероятностный):
┌─────────────────────────────────────────────┐
│ "60% шанс роста, но может упасть на 5%"     │
│ → Вкладывает только 10% (подходящий риск)   │
│ → Биткоин падает на 5%                      │
│ → Теряет только 0.5% сбережений             │
│ → Остаётся спокойным, ждёт                  │
│ → Биткоин восстанавливается, получает прибыль!│
└─────────────────────────────────────────────┘
```

## Попробуй сам!

### Простой код на Python

```python
# Очень простой пример вероятностного прогноза
import random

# Наш "прогноз" говорит: 60% шанс роста цены
up_probability = 0.60
expected_up = 0.02      # +2% если растёт
expected_down = -0.03   # -3% если падает

# Симулируем 100 дней торговли
total_money = 100  # Начинаем со 100$
kelly_fraction = 0.15  # Келли говорит ставить 15%
safe_fraction = kelly_fraction / 2  # Используем половину

for day in range(100):
    # Случайно симулируем рост или падение
    if random.random() < up_probability:
        # Цена выросла!
        change = expected_up
    else:
        # Цена упала
        change = expected_down

    # Наша прибыль/убыток
    profit = total_money * safe_fraction * change
    total_money += profit

print(f"После 100 дней: ${total_money:.2f}")
# Обычно вырастает до $115-130!
```

## Словарь терминов

| Термин | Простое объяснение |
|--------|-------------------|
| **Точечный прогноз** | Просто одно число (как "будет +22°C") |
| **Вероятностный прогноз** | Диапазон возможностей с шансами для каждого |
| **Квантиль** | Разделительная линия (как "95% цен будут ниже этого") |
| **Распределение** | "Форма горы" показывающая все возможности |
| **Калибровка** | Проверка, что твои уровни уверенности честные |
| **CRPS** | Оценка качества вероятностного прогноза |
| **Критерий Келли** | Формула "сколько нужно ставить?" |
| **VaR (Value at Risk)** | "Сколько максимум могу потерять с 95% уверенностью?" |
| **DeepAR** | Умная программа для вероятностных прогнозов |

## Главные выводы

1. **Неопределённость ценна** — Знать, чего ты НЕ знаешь, помогает принимать лучшие решения

2. **Диапазоны лучше чисел** — Прогноз "$95-$105" полезнее, чем просто "$100"

3. **Размер ставки важен** — Критерий Келли помогает ставить правильную сумму

4. **Будь честен в уверенности** — Откалиброванные прогнозы вызывают доверие и работают лучше

5. **Управление рисками встроено** — Вероятностные прогнозы автоматически включают меры риска
